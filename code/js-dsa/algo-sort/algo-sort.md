## Index

### Basic

1. [Selection Sort](selection-sort.js): Sorts an array by repeatedly finding the minimum element from unsorted part and putting it at the beginning.
2. [Bubble Sort](bubble-sort.js): Repeatedly compares adjacent elements and swaps them if they are in the wrong order.
3. [Insertion Sort](insertion-sort.js): Builds sorted array one item at a time by inserting each item in its proper place.

### Proficient

1. [Quick Sort](quick-sort.js): Picks an element as pivot and recursively partitions the array around the pivot and merges the sorted partitions.
2. [Merge Sort](merge-sort.js): Divides array in two halves recursively and then merges the sorted halves.
3. [Heap Sort](heap-sort.js): Builds a max heap from input array recursively and then swaps the root with last element to create the sorted array.
4. [Radix Sort](radix-sort.js): Sorts numbers digit by digit, starting from the least significant digit to the most significant digit,.
5. [Tim Sort](tim-sort.js): A hybrid sorting algorithm derived from merge sort and insertion sort.
