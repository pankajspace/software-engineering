## Inheritance

### Takeaways

1. Inheritance encourages a structured approach to **code organization, promoting clarity** and **maintainability.**
2. Inheritance allows new classes to **absorb attributes** and **methods from existing classes, enhancing code reusability.**
3. It facilitates the **extension** and **customization of existing code by deriving new subclasses.**
