## Python Installation
In Fedora Linux, you can install Python 3 using the following command:
```bash
sudo dnf install python3
```

## Jupyter Notebook Installation
To install Jupyter Notebook, first install pip
```bash
sudo dnf install python3-pip
```

Then install Jupyter using pip
```bash
pip3 install jupyter
```

## Running Jupyter Notebook
First navigate to the directory where you want to save your notebooks before starting Jupyter.
```bash
cd Workspace/software-engineering/codebasics/python/
jupyter notebook
```
This will open Jupyter Notebook in your default web browser. You can create new notebooks and start coding in Python right away!
