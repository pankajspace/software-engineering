# ECMAScript 5 (2009) to 2023 (ES14) 

Comprehensive guide to all major features introduced in ECMAScript (ES) from ES5 to the latest version with detailed examples.

## **ECMAScript 5 (ES5) - 2009**

### **1. Strict Mode**
Strict mode makes it easier to write "secure" JavaScript by throwing errors for unsafe actions, such as using undeclared variables.

#### **Example**:
```js
"use strict";
x = 10;  // Error: x is not defined
```

### **2. JSON Support**
ES5 introduced native JSON support with the `JSON.parse()` and `JSON.stringify()` methods.

#### **Example**:
```js
const jsonString = '{"name": "John", "age": 30}';
const obj = JSON.parse(jsonString);
console.log(obj.name);  // John

const newJson = JSON.stringify({ name: 'Alice', age: 25 });
console.log(newJson);  // {"name":"Alice","age":25}
```

### **3. Array Methods**
New methods for arrays were introduced, such as `forEach()`, `map()`, `filter()`, `reduce()`, `some()`, `every()`, and more.

#### **Example (map)**:
```js
const arr = [1, 2, 3];
const doubled = arr.map(x => x * 2);
console.log(doubled);  // [2, 4, 6]
```

### **4. `Object.keys()`, `Object.getOwnPropertyDescriptor()`, `Object.create()`**
These methods allow easier manipulation of objects.

#### **Example (`Object.keys()`)**:
```js
const obj = { name: 'John', age: 30 };
console.log(Object.keys(obj));  // ['name', 'age']
```

#### **Example (`Object.create()`)**:
```js
const person = {
  name: 'John',
  greet() {
    console.log(`Hello, my name is ${this.name}`);
  }
};

const john = Object.create(person);
john.greet();  // Hello, my name is John
```

#### **Example (`Object.getOwnPropertyDescriptor()`)**:
```js
const person = { name: 'John' };
const desc = Object.getOwnPropertyDescriptor(person, 'name');
console.log(desc.enumerable);  // true
```

### **5. `Function.prototype.bind()`**
Allows binding a function to a specific `this` value.

#### **Example**:
```js
const person = {
  name: 'John',
  greet() {
    console.log(`Hello, my name is ${this.name}`);
  }
};

const greetFunc = person.greet.bind(person);
greetFunc();  // Hello, my name is John
```

### **6. Property Getters and Setters**
ES5 introduced property accessors (getters and setters).

#### **Example**:
```js
const person = {
  firstName: 'John',
  lastName: 'Doe',
  get fullName() {
    return `${this.firstName} ${this.lastName}`;
  },
  set fullName(name) {
    [this.firstName, this.lastName] = name.split(' ');
  }
};

console.log(person.fullName);  // John Doe
person.fullName = 'Jane Smith';
console.log(person.fullName);  // Jane Smith
```

## **ECMAScript 6 (ES6) - 2015**

### **1. `let` and `const`**
- `let`: Block-scoped variable.
- `const`: Block-scoped constant.

#### **Example**:
```js
let x = 10;
const y = 20;
x = 30;  // Allowed
// y = 40;  // Error: Assignment to constant variable
```

### **2. Arrow Functions**
A shorter syntax for functions and lexical scoping for `this`.

#### **Example**:
```js
const add = (a, b) => a + b;
console.log(add(5, 3));  // 8
```

### **3. Template Literals**
Allows embedded expressions and multi-line strings using backticks (`` ` ``).

#### **Example**:
```js
const name = 'John';
const greeting = `Hello, ${name}!`;
console.log(greeting);  // Hello, John!
```

### **4. Default Parameters**
Allows setting default values for function parameters.

#### **Example**:
```js
function greet(name = 'Guest') {
  return `Hello, ${name}!`;
}
console.log(greet());  // Hello, Guest!
```

### **5. Destructuring Assignment**
Extract values from arrays or objects into distinct variables.

#### **Example (Array Destructuring)**:
```js
const [a, b] = [1, 2];
console.log(a);  // 1
```

#### **Example (Object Destructuring)**:
```js
const person = { name: 'John', age: 30 };
const { name, age } = person;
console.log(name, age);  // John 30
```

### **6. Spread and Rest Operators (`...`)**
- **Spread** expands arrays/objects.
- **Rest** collects multiple elements into an array.

#### **Example (Spread)**:
```js
const arr1 = [1, 2];
const arr2 = [...arr1, 3, 4];
console.log(arr2);  // [1, 2, 3, 4]
```

#### **Example (Rest)**:
```js
function sum(...numbers) {
  return numbers.reduce((a, b) => a + b, 0);
}
console.log(sum(1, 2, 3));  // 6
```

### **7. Classes**
A cleaner syntax for creating objects and handling inheritance.

#### **Example**:
```js
class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
  greet() {
    console.log(`Hello, my name is ${this.name}`);
  }
}

const john = new Person('John', 30);
john.greet();  // Hello, my name is John
```

### **8. Modules (`import`/`export`)**
Introduces modularity, allowing code export and import between files.

#### **Example**:
```js
// math.js
export function add(a, b) {
  return a + b;
}

// app.js
import { add } from './math.js';
console.log(add(2, 3));  // 5
```

### **9. Promises**
A cleaner way to handle asynchronous code.

#### **Example**:
```js
const fetchData = new Promise((resolve, reject) => {
  setTimeout(() => resolve('Data loaded'), 1000);
});

fetchData.then(data => console.log(data));  // Data loaded
```

### **10. `Symbol`**
A new primitive type that creates unique values, often used as object keys.

#### **Example**:
```js
const sym1 = Symbol('desc');
const sym2 = Symbol('desc');
console.log(sym1 === sym2);  // false
```

## **ECMAScript 7 (ES7) - 2016**

### **1. Exponentiation Operator (`**`)**
Provides a shorthand for exponentiation.

#### **Example**:
```js
console.log(2 ** 3);  // 8
```

### **2. `Array.prototype.includes()`**
Checks if an array contains a certain value.

#### **Example**:
```js
const arr = [1, 2, 3];
console.log(arr.includes(2));  // true
```

## **ECMAScript 8 (ES8) - 2017**

### **1. `async` and `await`**
A more readable way to work with Promises.

#### **Example**:
```js
async function fetchData() {
  const response = await fetch('https://api.example.com/data');
  const data = await response.json();
  return data;
}
fetchData().then(data => console.log(data));
```

### **2. `Object.values()` and `Object.entries()`**
- **`Object.values()`**: Returns an array of object values.
- **`Object.entries()`**: Returns an array of `[key, value]` pairs.

#### **Example (`Object.values()`)**:
```js
const person = { name: 'John', age: 30 };
console.log(Object.values(person));  // ['John', 30]
```

#### **Example (`Object.entries()`)**:
```js
console.log(Object.entries(person));  // [['name', 'John'], ['age', 30]]
```

### **3. String Padding (`padStart()` and `padEnd()`)**

#### **Example**:
```js
console.log('5'.padStart(3, '0'));  // '005'
console.log('5'.padEnd(3, '0'));    // '500'
```

### **4. Trailing Commas in Function Parameters**
You can now use trailing commas in function parameter lists.

#### **Example**:
```js
function add(a, b,) {
  return a + b;
}
```

## **ECMAScript 9 (ES9) - 2018**

### **1. Rest/Spread Properties for Objects**
The spread and rest operators now work with objects as well.

#### **Example (Spread)**:
```js
const person = { name: 'John', age: 30 };
const clone = { ...person };
console.log(clone);  // { name: 'John', age: 30 }
```

#### **Example (Rest)**:
```js
const { name, ...rest } = person;
console.log(rest);  // { age: 30 }
```

### **2. Asynchronous Iteration (`for await...of`)**

#### **Example**:
```js
async function* fetchData() {
  yield 'Data 1';
  yield 'Data 2';
  yield 'Data 3';
}

async function fetchAllData() {
  for await (const data of fetchData()) {
    console.log(data);
  }
}

fetchAllData();
// Data 1
// Data 2
// Data 3
```

### **3. `Promise.prototype.finally()`**
Executes code after a Promise is settled, regardless of its outcome.

#### **Example**:
```js
fetchData()
  .then(data => console.log(data))
  .catch(error => console.log(error))
  .finally(() => console.log('Cleanup'));
```

### **4. RegExp Enhancements**

#### **Named Capture Groups**:
```js
const regex = /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/;
const result = regex.exec('2023-08-19');
console.log(result.groups.year);  // 2023
```

## **ECMAScript 10 (ES10) - 2019**

### **1. `Array.prototype.flat()`**
Flattens nested arrays.

#### **Example**:
```js
const arr = [1, 2, [3, 4, [5]]];
console.log(arr.flat(2));  // [1, 2, 3, 4, 5]
```

### **2. `Array.prototype.flatMap()`**
Maps and flattens arrays in one step.

#### **Example**:
```js
const arr = [1, 2, 3];
const result = arr.flatMap(x => [x, x * 2]);
console.log(result);  // [1, 2, 2, 4, 3, 6]
```

### **3. `Object.fromEntries()`**
Transforms a list of key-value pairs into an object.

#### **Example**:
```js
const entries = [['name', 'John'], ['age', 30]];
const obj = Object.fromEntries(entries);
console.log(obj);  // { name: 'John', age: 30 }
```

### **4. String Trimming (`trimStart()` and `trimEnd()`)**

#### **Example**:
```js
console.log('   hello   '.trimStart());  // 'hello   '
console.log('   hello   '.trimEnd());    // '   hello'
```

## **ECMAScript 11 (ES11) - 2020**

### **1. Nullish Coalescing Operator (`??`)**
Returns the right-hand operand when the left-hand is `null` or `undefined`.

#### **Example**:
```js
let name = null;
console.log(name ?? 'Guest');  // Guest
```

### **2. Optional Chaining (`?.`)**
Allows safe access to deeply nested properties.

#### **Example**:
```js
const user = { address: { city: 'Paris' } };
console.log(user?.address?.city);  // Paris
console.log(user?.address?.zipcode);  // undefined
```

### **3. Dynamic Import**

#### **Example**:
```js
async function loadModule() {
  const { add } = await import('./math.js');
  console.log(add(2, 3));  // 5
}
loadModule();
```

## **ECMAScript 12 (ES12) - 2021**

### **1. Logical Assignment Operators (`||=`, `&&=`, `??=`)**

#### **Example**:
```js
let a = null;
a ??= 10;  // a = 10
console.log(a);  // 10
```

### **2. Numeric Separators**
Improves the readability of large numbers.

#### **Example**:
```js
const largeNumber = 1_000_000;
console.log(largeNumber);  // 1000000
```

---

## **ECMAScript 13 (ES13) - 2022**

### **1. Top-level `await`**
Allows using `await` outside of async functions in modules.

#### **Example**:
```js
const data = await fetch('https://api.example.com/data').then(res => res.json());
console.log(data);
```

### **2. `Array.prototype.findLast()` and `findLastIndex()`**

#### **Example**:
```js
const arr = [1, 2, 3, 4];
console.log(arr.findLast(n => n % 2 === 0));  // 4
console.log(arr.findLastIndex(n => n % 2 === 0));  // 3
```

---

## **ECMAScript 14 (ES14) - 2023**

### **1. `Array.prototype.toSorted()`**
Returns a sorted copy of the array without modifying the original.

#### **Example**:
```js
const arr = [3, 1, 4, 2];
const sortedArr = arr.toSorted();
console.log(sortedArr);  // [1, 2, 3, 4]
```

### **2. `Array.prototype.toReversed()`**
Returns a reversed copy of the array.

#### **Example**:
```js
const arr = [1, 2, 3];
console.log(arr.toReversed());  // [3, 2, 1]
```

### **3. `Array.prototype.with()`**
Replaces an element at a given index without modifying the original array.

#### **Example**:
```js
const arr = [1, 2, 3];
console.log(arr.with(1, 10));  // [1, 10, 3]
```
